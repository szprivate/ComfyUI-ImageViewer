Viewer Hotkeys

Left: Step Back
Right: Step Forward
Space: Play/Pause
Ctrl+Left: First Frame
Ctrl+Right: Last Frame
Ctrl+Enter: Queue Prompt
F: Toggle Fit Zoom
C: Cycle Compare Modes (off → split → contact)
R / G / B: Toggle Red, Green, Blue channel (press again for RGB)
Hold E + Drag: Adjust Exposure
Shift+Click Tabs: Select two for comparison
Shift+Click History: Compare two snapshots
Up/Down Arrows: Navigate history items
Ctrl-left drag on Timeline Select playback range

Roto Tool

+ Shape: then click to add points (drag to curve); click the first point to close
Drag a vertex: move it  ·  Single-click: show its tangents
Ctrl+drag a vertex: pull feather
Right-click a vertex: delete  ·  Alt+click an edge: insert point
Drag empty space: marquee-select points
With 2+ selected: drag inside = move, corners = scale (Shift = uniform), outside = rotate (Shift = 15°)
Middle-drag: pan

SAM3 Points Tool

Click: positive point  ·  Shift+click: negative point
Drag a point: move  ·  Right-click / Ctrl+click: delete
Middle-drag: pan

Click anywhere to close
