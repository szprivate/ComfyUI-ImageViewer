Keyboard Shortcuts

All hotkeys are active only when the mouse cursor is hovering over the viewer panel (except E-drag and Alt+Enter). Typing in an input or textarea field inside the parameter panel temporarily suspends hotkeys.

Quick reminder Press ? while hovering the viewer to open the built-in hotkey overlay without leaving the browser.

Frame Navigation

←
Step back one frame
→
Step forward one frame
Ctrl + ←
Jump to first frame (Rewind)
Ctrl + →
Jump to last frame (End)
Space
Play / Pause

History

↑
Go to previous history snapshot (newer)
↓
Go to next history snapshot (older)
Shift + click thumbnail
Select two history items for side-by-side comparison
Right-click thumbnail
Context menu — Copy Path / Remove snapshot

Tabs

1 – 9
Switch to tab 1–9 (in display order)
Shift + click tab
Select tab as comparison source; Shift-click another to open split view

Viewport

F
Toggle fit-to-screen zoom
S
Toggle tensor shape overlay
C
Toggle compare / split-view mode

Channels & Exposure

R
Isolate Red channel (press again to return to RGB)
G
Isolate Green channel
B
Isolate Blue channel
E + drag
Hold E and drag mouse horizontally to scrub exposure (−4 EV to +4 EV)
Right-click exposure control
Reset exposure to 0.0 EV

Timeline

Click timeline
Jump to frame at that position
Drag timeline
Scrub through frames
Ctrl + drag timeline
Define a playback sub-range (highlighted in orange)

Workflow

Alt + Enter
Queue prompt (run ComfyUI workflow) — works while hovering the viewer
?
Open / close the in-viewer hotkey help overlay

Mouse Interactions

Drag viewport
Pan image (when zoomed in beyond fit-to-screen)
Drag compare divider
Adjust the split position in comparison mode
Drag panel edge / corner
Resize the viewer panel
Drag tab bar (empty area)
Move the entire viewer panel
Drag panel resizer handle
Resize the parameter panel width
Drag history resizer
Resize the history strip height

Hotkey Conditions

ConditionEffect on hotkeys
Mouse outside viewer panelAll hotkeys disabled (except Alt+Enter)
Cursor inside an input / textareaHotkeys suspended while typing
Viewer undocked to popout windowHotkeys active in the popout window
