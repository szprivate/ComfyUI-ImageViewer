Panel Overview

Schematic of the viewer panel — orange = active/accent, grey = inactive regions.

Regions Explained

Tab Bar

The topmost strip holds one tab per image source. The active tab is highlighted orange. Tabs appear automatically when a bEpicSendToViewer node runs or when you open a folder via the file browser. Each tab can be:

  • Clicked to switch to it.
  • Dragged to reorder.
  • Closed via its × button.
  • Shift+clicked to select it as a comparison target (see Image Comparison).
  • Jumped to with number keys 1–9.

History Strip

A vertical thumbnail panel on the left side of the viewport. Every time a new image is generated for the active tab, a snapshot thumbnail is added here (up to 20 per tab). The selected snapshot has an orange border. Navigate with ↑ / ↓.

Right-clicking a thumbnail opens a context menu with options to Copy Path or Remove the snapshot.

Viewport

The central image display area. Supports:

  • Pan — click and drag when zoomed in.
  • Zoom — choose a preset (Fit, 100%, 75%, 50%) from the dropdown, or press F to toggle fit-to-screen.
  • Channel overlays — red, green, or blue isolation rendered with CSS SVG filters.
  • Compare layer — a second image layer appears in comparison mode.

Tensor Shape Overlay

A small cyan label in the top-left of the viewport displays the tensor dimensions of the current image, e.g. [1, 847, 1024, 3] or [8, 512, 512, 3] for a batch. Toggle visibility with the Shape button in the toolbar or press S.

Exposure & Channel Bar

Centred near the top of the viewport, this semi-transparent bar houses:

  • An Exposure slider (−4 EV to +4 EV).
  • An EV readout label (click to reset to 0).
  • A channel selector dropdown: RGB / R / G / B.

See Channels & Exposure for detailed usage.

Parameter Panel

The right-side panel mirrors the widget values of whichever ComfyUI node is currently selected on the canvas. Fields are live-editable. The panel can be:

  • Resized horizontally by dragging its left edge.
  • Locked to a specific node with the padlock button.
  • Switched to the left side with the dock button.

See Parameter Panel.

Resizing the Panel

Eight transparent resize handles sit on the edges and corners of the viewer. Drag any edge to resize in one axis; drag a corner for free-form resize. The panel remembers its position and size across page reloads.

Moving the Panel

Drag the tab bar (on an empty area, not a tab button) to reposition the entire floating panel anywhere on screen.
